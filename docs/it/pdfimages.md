# PDFimages

## Extract all images from PDF

`pdfimages -j [pdf] /tmp/out`

Will save images from PDF file `[pdf]` in files `/tmp/out-000.jpg`,
`/tmp/out-001.jpg`, etc.

[askubuntu](https://askubuntu.com/questions/150100/extracting-embedded-images-from-a-pdf)
